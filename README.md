## Introduction
The To-Do Web App is a task management system where users can create, update, delete, and mark tasks as completed. It includes an Admin Panel to manage users, view activity logs, and enforce access control.

## Features
 User Features
- User registration & authentication (NextAuth.js)
- Create, update, delete tasks
- Mark tasks as completed
- Categorize tasks with tags/labels
- Set task deadlines and reminders
- Drag-and-drop task prioritization

 Admin Panel Features
- View all users and their tasks
- Edit or remove inappropriate tasks
- View activity logs for moderation
- Manage user roles (admin, user)


## Technologies Used

- Framework: Next.js (T3 Stack)
- Language: TypeScript
- Backend: tRPC
- Database: PostgreSQL with Prisma/Drizzle ORM
- Hosting: Vercel / AWS
- State Management: React Query (via tRPC)
